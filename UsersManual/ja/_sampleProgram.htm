Star WebPRNT SDK

このSDKには、HTMLファイルとJavaScriptファイルがパッケージされています。

HTMLファイルは、WebブラウザーからWebPRNT機能を利用して印刷するまでの一連の記載方法を紹介するサンプルプログラムです。
実際に動かしながら、プリンターの制御方法を確認してください。

JavaScriptファイルは、スター精密が提供するWebPRNT機能組み込み用ライブラリファイルです。
変更は行わないでください。HTMLファイルから読み込んで使用します。

システム構築例

Star POSプリンター

※ 手順にマウスを合わせることで、イラストが変化します。

  1. Webサーバー(A)、Webブラウザーを表示する操作端末(B)、WebPRNT対応のStar POSプリンター / AsuraCPRNT (C)を、通信可能な環境に準備します。
  2. SDKにパッケージされているサンプルプログラム一式をWebサーバー(A)にコピーします。
  3. Webブラウザー(B)からサンプルプログラムを表示して、Star POSプリンター / AsuraCPRNT (C)のIPアドレスを設定します。
  4. 印刷データをStar POSプリンター / AsuraCPRNT (C)に送信します。
  5. Star POSプリンター / AsuraCPRNT (C)から印刷され、Webブラウザー(B)にプリンターステータスが戻ります。

 

AsuraCPRNT

※ 手順にマウスを合わせることで、イラストが変化します。

  1. WebPRNTに対応したAsuraCPRNT(Webサーバー)(A)、Webブラウザーを表示する操作端末(B)を、通信可能な環境に準備します。
  2. SDKにパッケージされているサンプルプログラム一式をAsuraCPRNT(A)の”/www/pages/"フォルダにコピーします。
  3. Webブラウザー(B)からサンプルプログラムを表示して、AsuraCPRNT(A)のIPアドレスを設定します。
  4. 印刷データをAsuraCPRNT(A)に送信します。
  5. AsruaCPRNT(A)から印刷され、Webブラウザー(B)にプリンターステータスが戻ります。

使用方法

  1. Webブラウザで公開したComparisonReceiptDesign.htmlを開き、JavaScript/Canvasのサポートに問題のないことを確認します。

    *"Your browser does not support JavaScript!"と表示された場合には、お使いのWebブラウザのJavaScript設定が無効になっていますので、JavaScriptを有効に変更してください。

    *"Your browser does not support Canvas!"と表示された場合には、Canvas機能に対応したWebブラウザへ変更してください。

  2. starWebPRNTエンドポイントURLをプリンタのIPアドレスに合わせて編集します。
  3. Send（API）ボタンでデバイスフォントによる印刷が行えることを確認します。
  4. Send（Canvas）ボタンでCanvas描画による印刷が行えることを確認します。
