<code>

Barcode reader result code Element

Attribute

NameDescriptionOmission
---

Value

Description
Barcode reader result code
  • 0 … Success
  • 1 … The read data of barcode data is nothing
  • 1001 … Barcode reader claim still has not be performed.
  • 1002 … Barcode reader is busy
    (In use with another device. Another device has already performed the claim start: Need to resend data)
  • 1003 … Barcode reader device is not connected on AsuraCPRNT or turn off.
